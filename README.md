# Healthcare Connect

## Project Overview
Healthcare Connect is a web-based healthcare assistance application developed using HTML, CSS, and JavaScript. 
It provides users with healthcare-related information and essential medical services through an interactive interface.

## Features
- AI Health Assistant
- Symptom Checker
- Disease Information
- Nearby Hospitals
- Blood Bank
- Emergency Services
- Vaccination Tracker
- Health Camps
- Health Awareness
- Government Schemes
- RMP Services
- Contact Support

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Font Awesome
- Google Fonts
